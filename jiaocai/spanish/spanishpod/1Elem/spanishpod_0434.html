Elementary - iPhone App (B0434)

A: Tengo una nueva aplicación en el iPhone.
I have a new application on my iPhone.
B: ¿Cuál?
Which one?
A: Es un diccionario español-inglés.
It is a Spanish-English dictionary.
B: Yo tengo uno, pero bastante cutre.
I have one, but it is very bad.
A: Pues éste es muy bueno y además es gratis.
Well, this one is very good and it's also free.

Key Vocabulary

aplicación application
diccionario dictionary
tener to have
bueno good
gratis free

Supplementary Vocabulary

teléfono phone
instalar to install
servir to be of use
tribanda tribanda
activar to activate
