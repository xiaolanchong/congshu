Intermediate - Zoo (C0210)

A: Je n`aime pas aller au zoo.
I don`t like going to the zoo.
B: Arrête, on n`y va qu` une fois par an.
Stop it, we only go there once a year.
A: Les animaux ont l`air tristes.
The animals look sad.
B: Mais essaie de voir les choses de manière pédagogique.
But try to see the educational side of it.
A: Tu aimerais vivre dans une cage?
Would you like to live in a cage?
B: Tu commences à m`énerver.
You start getting on my nerves.
A: En plus, ces animaux ont l`air d`avoir faim.
Moreover, these animals look like they`re hungry.
B: Lesquels?
Which ones?
A: Tous. Ces singes, ces girafes, ces éléphants, ces perroquets, ces gazelles.
All of them. These monkeys, these giraffes, these elephants, these parrots, these gazelles.
B: Et les ours, les tigres, les loups, les aigles, les lions...
And the bears, the tigers, the wolves, the eagles, the lions,...
A: Eux aussi, c`est un scandale!
Them too, it`s awful.
B: Ok, si tu veux vraiment les aider, je crois que j`ai une solution.
Ok, if you really want to help them, I think I have a solution.
A: Mais, non, qu`est-ce que tu fais?
Hey, no, what are you doing?
