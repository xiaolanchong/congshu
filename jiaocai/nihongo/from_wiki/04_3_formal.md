# わけ

= Как самостоятельное слово =

== Причина, обстоятельство ==

どういうわけで日本に留学したんですか。

私が断ったのには、深いわけがあります。

詳しいわけを話して下さい。

== わけない легко ==

彼にはラジオの修理なんてわけないよ。

こんな数学問題なんかわけなく解けるよ。

== わけがわからない непонятный ==

何だがわけのわからない字が書いてある。

何が言いたいのか、わけがわからない。
