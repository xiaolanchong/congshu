Intermediate - Formatting a Word Document (C1957)

A: 小于，我看了你编辑的文件，有几个地方要修改。
Xiǎo Yú, wǒ kàn le nǐ biānjí de wénjiàn, yǒu jǐ ge dìfang yào xiūgǎi.
Xiao Yu, I took a look at the document you edited. There are a few places that need to be revised.
A: 这是会议文件，标题的字体要正式一些。
zhè shì huìyì wénjiàn, biāotí de zìtǐ yào zhèngshì yīxiē.
This is a document for a meeting, so the title font should be a bit more formal.
B: 那我改成黑体吧。
nà wǒ gǎi chéng hēitǐ ba.
I'll make it Hei, then.
A: 嗯。这些数据做成表格会很清楚。
ng4. zhèxiē shùjù zuò chéng biǎogé huì hěn qīngchǔ.
Good. This data would be really clear as a table.
B: 好的。
hǎo de.
OK.
A: 还有，这个地方的格式很奇怪。
háiyǒu, zhè ge dìfang de géshì hěn qíguài.
Also, the formatting here is really weird.
B: 这些是从别的地方复制过来的，不好修改。
zhèxiē shì cóng bié de dìfang fùzhì guòlai de, bù hǎo xiūgǎi.
These are copied from somewhere else. I can't change them.
A: 可以用格式刷。选中上面的几句话，点一下格式刷，再刷一下下面的。
kěyǐ yòng géshìshuā. xuǎn zhòng shàngmian de jǐ jù huà, diǎn yīxià géshìshuā, zài shuā yīxià xiàmian de.
You can use the format painter. Select the sentences up top, click on the format painter, and then select the lower section.
B: 哎，这个刷子挺好用的。
āi, zhè ge shuāzi tǐng hǎoyòng de.
Hey, this brush is quite useful.
A: 还有这里，这些不重要，可以去掉。
háiyǒu zhèlǐ, zhèxiē bù zhòngyào, kěyǐ qùdiào.
Also, these parts here aren't important. You can get rid of them.
B: 好。
hǎo.
OK.
A: 你修改一下，改好后打印十份。
nǐ xiūgǎi yīxià, gǎi hǎo hòu dǎyìn shí fèn.
Fix it up and then when it's done print out ten copies.
B: 好，我马上去。
hǎo, wǒ mǎshàng qù.
OK, I'll be over there right away.
